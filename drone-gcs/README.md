# DOLUCK GCS — ドローン地上管制UI

ドローンを飛行させて、地上の追尾アンテナからのテレメトリ・コマンドをPCに表示する、
SpaceXのミッションコントロール風UI。

## 構成

```
[ドローン] --MAVLink/無線--> [追尾アンテナ + 受信機] --USB/UDP--> [PC]
                                                                  ├ bridge.py (MAVLink→WebSocket変換)
                                                                  └ index.html (ブラウザUI) ← ws://localhost:8765
```

- **index.html** — UI本体。1ファイル完結・依存なし。ブラウザで開くだけで動く
- **bridge.py** — 実機接続用。MAVLinkテレメトリをJSONに変換してWebSocketで配信し、UIからのコマンドをMAVLinkに変換する

## まず見る(デモモード)

`index.html` をブラウザで開くだけ。実機リンクがないときは内蔵フライトシミュレーションに
自動フォールバックし、カウントダウン(T−10s)→離陸→ウェイポイント巡回2周→RTL→着陸のデモ
ミッションが流れる。右上バッジが `SIM` 表示になる。

コマンドボタン(HOLD / RESUME / RTL / LAND、着陸後は ARM / LAUNCH)はシミュレーションにも効く。

## 実機接続(LIVEモード)

```bash
pip install pymavlink websockets

# テレメトリ無線(SiK 915/433MHz等)がUSBシリアルの場合
python bridge.py --mav /dev/ttyUSB0 --baud 57600

# SITLやWiFiテレメトリ(UDP)の場合
python bridge.py --mav udp:0.0.0.0:14550
```

ブリッジ起動後に `index.html` を開くと自動で `ws://127.0.0.1:8765` に接続し、
バッジが `LIVE` (緑)になる。追尾アンテナのAZ/ELはブリッジ側でホーム位置からの
方位角・仰角として計算している(実アンテナのローテーター制御に流用可)。

## UIの見どころ

| 要素 | 内容 |
|---|---|
| ターゲットレティクル | 機体を追尾するリング+ブラケット。左弧=速度 / 右弧=高度、データタグ付き |
| クロック | 上部中央にT+クロックとフェーズ(`巡航 // CRUISE`) |
| 円形レーダー | 航跡・ウェイポイント・追尾アンテナビーム・回転スイープ。AZ/EL/RSSI/LQ併記 |
| イベントストリーム | 左下5行、古い行からフェードアウト |
| コマンド | `[ 起動 ARM ] [ 発進 LAUNCH ] [ 保持 HOLD ] [ 再開 RESUME ] [ 帰投 RTL ] [ 着陸 LAND ]` |
| タイムライン | 下部中央の極細ライン(LIFTOFF→CRUISE→LAP 1→LAP 2→RTL→TD) |
| 詳細レイヤー | [詳細 DETAIL] で姿勢儀・アンテナ天球図・チャート等を展開(`?detail=1`) |

## デザインメモ(v3: 攻殻機動隊スタイル)

情報量を絞ったミニマルHUD。SAC系のアンバー細線ワイヤーフレームの世界観。

- **ターゲットレティクル**: 画面中央の機体を囲む回転ダッシュリング+コーナーブラケット。左弧=速度、右弧=高度、リーダーライン付きデータタグ(`UAV-01 // ALT … // GS …`、`追尾同期` ブリンク)
- **円形レーダー**(左下): 地上局中心の航跡+ウェイポイント+追尾アンテナビーム+回転スイープ(残光付き)。下に AZ/EL/RSSI/LQ のマイクロテキスト
- **最小限のクローム**: 上部中央にT+クロックとフェーズ(`巡航 // CRUISE`)、左下に5行のイベントストリーム(古い行はフェード)、右下に `[ 保持 HOLD ]` 形式のブラケットコマンド
- **[詳細 DETAIL] トグル**で全データ展開(飛行データ・電源/GPS・姿勢儀・アンテナ天球図・テレメトリチャート)。`?detail=1` で最初から展開
- **演出**: コールアウトはRGBずれのグリッチアニメーション+日英二段表示(`LIFTOFF / リフトオフ`)、走査線オーバーレイ、離陸シェイク(prefers-reduced-motion対応)
- 日本語ラベル(待機/上昇/巡航/帰投/着陸 等)と英語の併記
- D-DINフォント埋め込み(OFL-1.1)、音声トグル(WebAudio、デフォルトOFF)は継続
- チャート系列色 ティール `#1CA88E`・アンバー `#B07E3B` — ダーク面でCVD分離・コントラスト検証済み

過去バージョン: v1(パネル型ダッシュボード)→ v2(SpaceX中継スタイル)は git履歴参照。
