# 億り人 OKURIBITO 〜兆への道〜

お金が貯まっていく高揚感に全振りしたインクリメンタル(クリッカー)ゲームのプロトタイプ。

**▶ 遊ぶ: https://ryo208.github.io/idea/game/**

## 高揚感の設計

| 仕掛け | 内容 |
|---|---|
| 単位の繰り上がり | 円 → 万 → 億 → 兆 → 京。日本語の数詞で桁が上がる快感 |
| 肩書きの昇格 | 小銭拾い → バイト戦士 → 貯金の鬼 → 成金 → **億り人** → 大富豪 → 財閥総帥 → 兆の支配者 → 京を見た者 |
| 節目の判子スタンプ | 一万円/一億円/一兆円…到達で「祝」の朱印がドンと押され、札吹雪+ファンファーレ |
| 音 | タップでチャリン、購入でチャチャーン(WebAudioでリアルタイム生成) |
| 物理コイン | タップ位置から小判が弾け飛ぶ |
| 投資メニュー | 自販機 → ラーメン屋台 → コンビニFC → 全国チェーン → ITスタートアップ → タワマン → 宇宙ベンチャー → 造幣局。前段を買うと次が解放 |
| オフライン収益 | 閉じている間も投資が稼ぐ(最大8時間)。「おかえりなさい」で受け取り |
| 自動セーブ | localStorage、5秒ごと |

## デザイン

紙幣の世界観に振り切ったワンテーマ: 深緑黒(お札のインク)×金箔×和紙クリーム、節目だけ判子朱。金額数字はD-DIN(OFL)、見出し・肩書きは明朝(紙幣の彫刻感)。

## Steamへの道(ロードマップ)

1. **プロトタイプ検証(いまここ)** — Webで遊んで「貯まる快感」の芯を確認。友達に配って反応を見る
2. **Steamworks開発者登録** — https://partner.steamgames.com/ から。アプリ登録料 **$100/本**(売上$1,000超で返金)。銀行口座と税務情報(W-8BEN)の登録が必要
3. **デスクトップ化** — Webのまま **Electron か Tauri** でexe化(Steamのインクリメンタル系はWeb技術製が多数)。または Godot 4 に移植
4. **Steamworks SDK組み込み** — 実績(「初めての一億円」等はこのゲームと相性抜群)、クラウドセーブ、オーバーレイ
5. **ストアページ公開** — カプセル画像・スクショ5枚・トレーラー・説明文。**リリースの2週間以上前に「近日登場」ページ公開が必須**。ウィッシュリスト集めが命
6. **デモ版 + Next Fest** — 体験版を出してSteam Next Festに参加するとウィッシュリストが伸びやすい
7. **審査 → リリース** — ページ審査・ビルド審査(各数日)。価格はこのジャンルだと¥400〜980 か 基本無料が主流
